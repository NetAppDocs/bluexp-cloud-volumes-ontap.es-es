=== Soporte para las regiones de China para Cloud Volumes ONTAP 9.13.1 en AWS

Ahora puede implementar Cloud Volumes ONTAP 9.13.1 en AWS en las regiones de China. Esto incluye compatibilidad con puestas en marcha de un solo nodo y de alta disponibilidad. Después de implementar Cloud Volumes ONTAP 9.13.1, puede actualizarla a versiones posteriores. Solo se admiten las licencias que se han adquirido directamente a través de NetApp.

Para conocer la disponibilidad regional, consulte la https://bluexp.netapp.com/cloud-volumes-global-regions["Mapas de regiones globales para Cloud Volumes ONTAP"^].
